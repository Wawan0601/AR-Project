Panduan Setup Supabase (Lakukan Sekali)

1
Buat akun Supabase

Buka supabase.com → Sign up gratis dengan GitHub.

2
Buat project baru

Klik "New Project" → beri nama → pilih region Asia (Singapore) → set password database → Create.

3
Buat tabel produk

Buka menu Table Editor → "New Table" → nama tabel: produk. Lalu tambah kolom berikut:

id → uuid, primary key, default: gen_random_uuid()
barcode → text, NOT NULL, unique
nama → text, NOT NULL
deskripsi → text
dimensi → text
berat → text
harga → text
model_url → text
poster_url → text
created_at → timestamptz, default: now()
4
Buat Storage Bucket

Buka menu Storage → "New Bucket" → nama: produk-assets → centang Public bucket → Create.

5
Aktifkan Row Level Security (RLS) — opsional tapi direkomendasikan

Di Table Editor → tabel produk → RLS → Enable. Lalu tambah policy: Allow all for anon untuk SELECT, INSERT, UPDATE, DELETE. (Untuk production, batasi hanya SELECT untuk publik.)

6
Isi js/supabase-config.js

Buka Settings → API di Supabase. Copy:

url: 'https://xxxx.supabase.co'
anonKey: 'eyJhbGci...'

Paste ke file js/supabase-config.js di repo GitHub Anda.

7
Deploy ulang ke GitHub

Commit & push perubahan → GitHub Pages otomatis update dalam ~30 detik.
